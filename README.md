## 📋 Project Overview
This project is a professional desktop application developed as part of the **CodeAlpha Artificial Intelligence Internship**. It provides real-time language translation across multiple supported languages and features an intuitive Graphical User Interface (GUI). 

To enhance accessibility and user experience, the application also includes intelligent Text-to-Speech (TTS) integration and clipboard management.

## ✨ Key Features
- **🌍 Multi-Language Support:** Translates text between dozens of globally recognized languages.
- **🎙️ Text-to-Speech (TTS):** Converts translated text into high-quality audio playback using Google's TTS engine.
- **🧠 Smart Capability Filtering:** Automatically filters the language dropdown menu to only display languages that support *both* text translation and audio playback.
- **📋 One-Click Copy:** Seamlessly copies the translated output directly to the system clipboard.
- **🖥️ Clean GUI:** Built with Tkinter for a lightweight, responsive, and user-friendly desktop experience.

## 🛠️ Tech Stack & Libraries
- **Language:** Python 3
- **GUI Framework:** Tkinter (Standard Library)
- **Translation Engine:** `deep_translator` (Google Translate API)
- **Audio/Speech:** `gTTS` (Google Text-to-Speech) & `playsound`
